# 介绍

一个跨平台的 javascript 通信脚本

## 支持

- Vue
- React
- Angular
- UNI-APP
- 小程序

## 安依赖

```
npm install cz-event-bus

or

yarrn add cz-event-bus
```

## 用起来

```
import CZEventBus from 'cz-event-bus'
```
